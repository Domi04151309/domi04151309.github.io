--- title: Transparent Taskbar ---

Make Taskbar Transparent in Windows 10

In this tutorial you will learn how to make the Windows 10 taskbar transparent.

First download and install Classic Shell


Then open "Classic Start Menu Settings


Check "Show all settings" and select "Controls"
To get back the original Windows 10 start menu apply these settings


Now you only have to go to "Taskbar", check "Customize taskbar" and "Transparent"


Now your taskbar is transparent
