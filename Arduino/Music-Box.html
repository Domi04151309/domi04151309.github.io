Arduino Music Box

This is a little project which has LEDs blinking to music played by a speaker. When you push the buttons songs will play.

The sketch uses the Notes Library!

What You Need

A list of what you need:

  • An Arduino Uno
  • A Speaker
  • 8 LEDs
  • 8 Resistors
  • 4 Buttons
  • A Lot of Wires
  • The Code

Where Do You Get the Code?

You can download the code from here!

Why Does Not It Work?

You probably forgot to install the Notes Library from here.
If that is not the case please contact me.
